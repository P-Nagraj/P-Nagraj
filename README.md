### 🚀 Featured Projects

#### 🔹 [Todo Application API](https://github.com/P-Nagraj/Todo-Application-Api)
A full CRUD REST API built with **Node.js & Express** for managing tasks. Supports fetching todos with filters (status/priority), creating new todos, partial updates (status, priority, or task text) via `PUT`, and deletion by ID — built with CommonJS modules and a clean Express router structure.
> **Stack:** Node.js · Express.js · SQLite

#### 🔹 [Comments App](https://github.com/P-Nagraj/Commentsapp)
A React application built using **Class Components** to manage user comments end-to-end. Covers state management with `setState`, controlled forms, conditional rendering, and array methods like `map`/`filter` to render and manage dynamic comment lists with a responsive UI.
> **Stack:** React.js (Class Components) · CSS
> **Learning focus:** State management, event/form handling, props & component communication
> **Planned enhancements:** Edit comments, local storage persistence, backend integration, authentication, reply-to-comment feature

#### 🔹 [User Authentication Backend](https://github.com/P-Nagraj/User-authentication-backend-project)
A secure authentication API built with **Node.js & Express**, handling login and password-change flows with proper validation and status codes — including invalid username/password handling, short-password checks, and successful login/update confirmations.
> **Stack:** Node.js · Express.js · SQLite
